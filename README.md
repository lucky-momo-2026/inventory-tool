# 📦 在庫管理ツール

CSVファイルから在庫データを読み込み、在庫切れ・在庫少の商品を自動でピックアップして結果をCSVに書き出すPythonスクリプトです。

---

## 📋 機能一覧

- 在庫切れ（stock = 0）の商品を一覧表示
- 在庫少の商品を一覧表示（デフォルト：5以下、コマンドライン引数で変更可能）
- 仕入先ごとの要補充件数を集計・表示
- 分析結果を `stock_result.csv` に書き出し（出力日時・合計件数・仕入先集計つき）
- 仕入先ごとに補充対象商品をCSVファイルで個別出力（例：`富士文具.csv`）

---

## 🗂️ ファイル構成

```
inventory-check/
├── stock_check.py        # メインスクリプト
├── data.csv              # 入力データ
├── stock_result.csv      # 出力：在庫チェック結果
├── 富士文具.csv           # 出力：仕入先ごとの補充対象商品
├── サクラ商事.csv         # 出力：仕入先ごとの補充対象商品
├── 東京オフィス.csv       # 出力：仕入先ごとの補充対象商品
└── オフィス販社.csv       # 出力：仕入先ごとの補充対象商品
```

---

## 📦 必要なライブラリ

標準ライブラリのみ使用のため、インストール不要です。

---

## 📄 入力CSVの形式

```csv
item,category,stock,supplier
ノート,A,15,サクラ商事
ペン,B,3,富士文具
消しゴム,B,0,富士文具
ファイル,A,8,東京オフィス
コピー用紙,A,2,サクラ商事
ホッチキス,C,12,オフィス販社
付箋,B,1,富士文具
```

---

## 🚀 使い方

```bash
# デフォルト（5以下）
python stock_check.py

# しきい値を変更する場合（例：3以下）
python stock_check.py 3
```

### 出力例（ターミナル）

```
在庫切れの商品の一覧
------------------
在庫切れ： 消しゴム /仕入れ先： 富士文具

在庫が少ない商品の一覧
------------------
在庫少： ペン 3 /仕入先： 富士文具
在庫少： 消しゴム 0 /仕入先： 富士文具
在庫少： コピー用紙 2 /仕入先： サクラ商事
在庫少： 付箋 1 /仕入先： 富士文具

------------------
```
